## 函数声明和函数表达式
### 函数声明
```
fn();//不会报错
function fn(){};
```

## 函数表达式
```text
fn1();//会报错,因为在执行之前他并不是一个函数.结果是undefined.
//也就是 var fn1 = undefined;
var fn1 = function(){};


就相当于:
var a = undefined;
var a =100;
也可以用下面的测试;

console.log(a);
var a =100;
```

### 全局声明
1.<script></script>就是把变量提前.再去执行这个函数.

2.函数里面:函数即将执行,会把变量提前.


```text
fn('zhangsan');
function fn(name)
{
    console.log(name);
};


fn('zhangsan');//这是在全局的环境变量中.
//会把变量提前,函数声明提前.
function fn(name)
{
    //在代码执行之前,this就已经确定值了.
    console.log(this);
    console.log(arguments);//参数的集合体
    age =20;
    //程序执行之前对age这个参数进行占位.
    console.log(name,age);
    
    //这是在函数的环境变量中.
    bar(100);
    function bar(num){
        console.log("你没有"+num+"个bar")
    };
    var age;
};
```
